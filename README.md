## Dependencies
- make >= 3.82
- python3
- ffmpeg

## Instructions
||Command|
|---|---|
|Help|`make`|
|Install python dependencies|`make install`| 
|Run test script|`make test`|
|Run interactive downloader|`make shell`|
|Run downloader on list of urls|`make shell LIST=[filename]`|
|Delete garbage|`make clean`|

Mp3s go to `./out`
